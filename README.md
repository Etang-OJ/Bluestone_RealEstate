\# Bluestone Real Estate ML API



\## Project Overview

This project deploys two machine learning models as a FastAPI service:



\- Rental demand prediction model

\- Sales price prediction model



The API provides prediction endpoints for both rental and sales scenarios.



---



\## Files in the Project



\- `app.py` → FastAPI application

\- `requirements.txt` → Python dependencies

\- `Output/` → saved trained models and metadata

\- `README.md` → project documentation

\- `.gitignore` → files ignored by Git



---



\## Model Files



The following files are stored in the `Output/` folder:



\- `rental\_demand\_model.joblib`

\- `sales\_price\_model.joblib`

\- `feature\_metadata.joblib`
